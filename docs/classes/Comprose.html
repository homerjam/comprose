Class Comprose

This class creates a Comprose instance and defines the default dataset and locale to be used by all methods.

example
import { Comprose } from 'comprose';

const comprose = new Comprose({
  dataset: 'faker',
  locale: 'en',
});

comprose.personName();
// John Smith
